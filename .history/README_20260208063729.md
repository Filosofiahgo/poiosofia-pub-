# poiosofia-pub-
Repositorio para textos, informes y publicaciones poiosóficas
## Estructura del Proyecto

```
/publicar        → Archivos Markdown listos para publicar
/borradores      → Trabajo en curso
/plantillas      → Plantillas Markdown para tus tipos de texto
```

- **publicar/**: Contenido finalizado y que ha pasado revisión, listo para ser publicado o compartido
- **borradores/**: Documentos en desarrollo, notas y trabajo en progreso
- **plantillas/**: Estructuras y formatos reutilizables para diferentes tipos de documentos
