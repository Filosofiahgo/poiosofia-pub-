# poiosofía-pub-
Repositorio para textos, informes y
publicaciones poiosóficas.

## Estructura del Proyecto

```tree
poiosofía-pub-/
├── Arts-Performing/     → Guiones, notas de performance, medios y expresiones artísticas
├── Critical-Thinking/   → Ensayos, críticas, bitácoras y análisis
├── Theory-Practice/     → Textos filosóficos, investigación y desarrollos teóricos
├── borradores/          → Trabajo en curso, ideas en desarrollo
├── plantillas/          → Plantillas Markdown reutilizables para cada tipo de texto
├── publicar/            → Archivos Markdown listos para publicar
├── CONTRIBUTING.md      → Guía del flujo de trabajo editorial
└── README.md            → Este archivo
```

## Flujo de Trabajo Editorial

1. **Crear** — Copia una plantilla de `plantillas/` a `borradores/` y comienza a escribir.
2. **Iterar** — Desarrolla, revisa y madura el texto en `borradores/`.
3. **Clasificar** — Mueve el borrador a la carpeta temática correspondiente (`Arts-Performing/`, `Critical-Thinking/` o `Theory-Practice/`).
4. **Publicar** — Cuando el texto esté finalizado, cópialo a `publicar/`.

Consulta [CONTRIBUTING.md](CONTRIBUTING.md) para detalles completos del flujo de trabajo.

## Plantillas Disponibles

| Plantilla | Archivo | Uso |
|-----------|---------|-----|
| Análisis | `plantillas/plantilla-analisis.md` | Análisis crítico multi-dimensional |
| Informe | `plantillas/plantilla-informe.md` | Informes estructurados con metodología |
| Nota Breve | `plantillas/plantilla-nota-breve.md` | Ideas rápidas y observaciones cortas |
| Observaciones | `plantillas/plantilla-observaciones.md` | Observaciones insurgentes |
| Texto Creativo | `plantillas/plantilla-texto-creativo.md` | Ensayos, poesía y escritura creativa |
